## 📋 Sobre o Projeto

O PetLembra é um sistema completo para gerenciamento de pets e agendamentos para clínicas veterinárias e pet shops. Este repositório contém o **backend** da aplicação.

### Funcionalidades Principais

| Funcionalidade | Descrição |
|----------------|-----------|
| 👥 **Clientes** | Cadastro, edição, listagem e exclusão de clientes |
| 🐕 **Pets** | Cadastro de pets associados a clientes |
| 📅 **Agendamentos** | Agendamento de serviços (banho, tosa, consulta, vacina) |
| 📊 **Dashboard** | Estatísticas e agendamentos do dia |
